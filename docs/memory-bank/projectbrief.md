# Proje Özeti: SubTracker

SubTracker, kullanıcıların aboneliklerini (Netflix, Spotify vb.) takip etmelerini sağlayan, gizlilik odaklı (local-first), çevrimdışı çalışabilen (PWA) modern bir web uygulamasıdır.

## Temel Hedefler

- **Gizlilik:** Tüm veriler kullanıcının cihazında (LocalStorage) saklanır. Sunucu yoktur.
- **Farkındalık:** Aylık/Yıllık toplam harcamaları ve yaklaşan ödemeleri net bir şekilde gösterir.
- **Erişilebilirlik:** Mobil öncelikli (Mobile First) tasarım ve PWA desteği ile her yerde çalışır.
- **Hız:** Modern web teknolojileri (Vite, React 19) ile anlık tepki verir.

## Çekirdek Özellikler

1. Abonelik Ekleme/Düzenleme/Silme
2. Yaklaşan Ödemeler Takvimi
3. Harcama Analitiği (Kategori bazlı)
4. Bildirimler (Tarayıcı destekli)
5. Veri Yedekleme (JSON Export/Import)
