# PWA Icons

## Required Icons:

- `icon-192.png` - 192x192px (Android home screen)
- `icon-512.png` - 512x512px (Android splash screen + maskable)
- `apple-touch-icon.png` - 180x180px (iOS home screen)

## Design Specs:

- **Theme color:** #0f766e (Teal-700)
- **Background:** #0f172a (Slate-950)
- **Style:** Minimalist, modern, calendar/bell icon
- **Format:** PNG with transparency
